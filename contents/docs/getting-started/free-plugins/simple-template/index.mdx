**Simple Template** is a WordPress Plugin for applying a gutenberg design to be a single post template. You can create a custom query loop too!

Simple Template requires <a href="https://studioinspirasi.com/product/dynamic-data" target="_blank">Dynamic Data</a> plugin to work properly. So, please install both of the plugin.

##  Features

### Create Single Post Template

With Simple Template, you can create a custom single post template with conditions:
- `{Specific Custom Post Type}`: The custom single template will be applied for entire post in specific custom post type. Example: `book`, `vehicle`, etc.
- `{Specific Post}`: The custom single post template will be applied for specific post. Example: The posts with the id `87`, `433`, etc

### Create Custom Query Loop 

With Simple Template, you can create a custom query loop that being displayed in the frontend by using shortcode `{query-loop query=98 template=76}`.
- `query`: the id of the query
- `template`: the id of the template that will be used by the query.
