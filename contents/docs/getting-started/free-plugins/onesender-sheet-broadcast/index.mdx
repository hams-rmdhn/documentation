**Onesender Sheet Broadcast** is a Simple WordPress Plugin to broadcast Whatsapp Messages using OneSender API with the contact form comes from Google Sheet.

Just Simply go to the `broadcast page` in the dashboard, you can start broadcasting the messages to the contacts.

##  General

###  Install

<Stepper>
  <StepperItem title="Step 1: Download">
    Download the plugin in <a href="https://studioinspirasi.com/product/onesender-sheet-broadcast" target="_blank">Studio Inspirasi</a>, so you will get the zip file of the recent plugin update.
  </StepperItem>
  <StepperItem title="Step 2: Upload & Install">
    Upload and Install the plugin to your WordPress by navigating to your wordpress dashboard menu in: `Plugins` > `Add New` > `Upload Plugin` > `Choose File` > `Install Now`.
  </StepperItem>
  <StepperItem title="Step 3: Activate">
    After successfully install the plugin, click the `Activate` link in the process.
  </StepperItem>
</Stepper>

### Dynamic Data

To use the dynamic data in Google Sheet, you should create a google sheet that contains at least 4 columns filled with data. Here is the details:

- `Column 1`: should be a valid phone number
- `Column 2`: can be anything, you can get the data with `{detail1}` shortcode
- `Column 3`: can be anything, you can get the data with `{detail2}` shortcode
- `Column 4`: can be anything, you can get the data with `{detail3}` shortcode

##  Use Cases

**Onesender Sheet Broadcast** developed and used by Studio Inspirasi to make an easiest way to interact with the Onesender API. The main function of this plugin is to broadcast a blast messages, so the plugin is suitble if you're:
