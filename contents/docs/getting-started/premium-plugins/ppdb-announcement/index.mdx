**Affiliate Partners for Indonesian Marketplace** adalah plugin untuk menampilkan daftar tombol pembelian ke marketplace yang ada di Indonesia.

Penggunaannya cukup mudah, cukup gunakan shortcode `[aff-partner id=78125]` untuk menampilkan daftar tombol marketplace di frontend.

##  Umum

###  Instalasi

<Stepper>
  <StepperItem title="Step 1: Unduh">
    Dapatkan plugin di <a href="https://studioinspirasi.com/product/affiliate-partners-for-indonesian-marketplace/" target="_blank">Studio Inspirasi</a>, maka Anda akan mendapatkan file plugin berbentuk `.zip`.
  </StepperItem>
  <StepperItem title="Step 2: Unggah dan Instal">
    Unggah plugin ke website Anda melalui menu dashbor: `Plugins` > `Add New` > `Upload Plugin` > `Choose File` > `Install Now`.
  </StepperItem>
  <StepperItem title="Step 3: Aktifkan">
    Setelah berhasil melakukan instalasi, klik `Activate` untuk mengaktifkan plugin.
  </StepperItem>
</Stepper>
