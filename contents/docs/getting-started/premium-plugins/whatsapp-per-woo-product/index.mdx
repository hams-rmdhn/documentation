**Whatsapp Per Woo Product** is a plugin used to display whatsapp button on every woocommerce product with different whatsapp number direction.

This Plugin is suitable for:
- You who create an online store together with others
- Those of you who have an online store with sellers of every different product
- You have a customer base that likes to checkout using whatsapp

##  General

###  Install

<Stepper>
  <StepperItem title="Step 1: Download">
    Get the plugin by purchasing in <a href="https://studioinspirasi.com/product/tabler-login-for-wordpress/?lang=en" target="_blank">Studio Inspirasi</a>, so you will get the zip file of the recent plugin update.

  <StepperItem title="Step 2: Upload & Install">
    Upload and Install the plugin to your WordPress by navigating to your wordpress dashboard menu in: `Plugins` > `Add New` > `Upload Plugin` > `Choose File` > `Install Now`.
  </StepperItem>
  <StepperItem title="Step 3: Activate">
    After successfully install the plugin, click the `Activate` link in the process.
  </StepperItem>
</Stepper>

### Sellers

After activation, you will find a new submenu under WooCommerce > Sellers. Here you can enter the name and WhatsApp number of the seller for each product.

### Seller Per Product

When you add a product, you will see a Seller field to determine who is selling this product. The data that appears here is a result of the input on the previous Sellers page.
