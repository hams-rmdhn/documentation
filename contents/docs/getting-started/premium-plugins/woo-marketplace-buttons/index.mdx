**Woo Marketplace Buttons** adalah sebuah plugin untuk menampilkan tombol menuju sebuah produk ke marketplace tertentu.

Fitur Utama:

- Menampilkan 6 marketplace besar di Indonesia: tokopedia, shopee, bukalapak, lazada, blibli, tiktok
- Formulir input menuju marketplace bisa didapati ketika Anda sedang mengedit sebuah produk di tab marketplace
- Plugin sangat ringan dan tidak bloated (yaitu tidak dipenuhi fitur yang mubazir sehingga memberatkan server)

##  Umum

###  Instalasi

<Stepper>
  <StepperItem title="Step 1: Unduh">
    Dapatkan plugin dengan membeli di <a href="https://studioinspirasi.com/product/woo-marketplace-buttons/" target="_blank">Studio Inspirasi</a>. Setelah pembelian berhasil, Anda dapat mendapatkan file `.zip` di <a href="https://studioinspirasi.com/my-account/downloads/" target="_blank">Halaman Dasbor</a>.
  </StepperItem>
  <StepperItem title="Step 2: Unggah dan Instal">
    Unggah plugin ke website Anda melalui menu dashbor: `Plugins` > `Add New` > `Upload Plugin` > `Choose File` > `Install Now`.
  </StepperItem>
  <StepperItem title="Step 3: Aktifkan">
    Setelah berhasil melakukan instalasi, klik `Activate` untuk mengaktifkan plugin.
  </StepperItem>
</Stepper>

###  Lisensi

<Stepper>
  <StepperItem title="Step 1: Dapatkan Lisensi">
    Setelah pembelian plugin Anda berhasil, Anda akan mendapatkan lisensi di <a href="https://studioinspirasi.com/my-account/license-key/" target="_blank">Halaman Dasbor</a>. Lisensi adalah karakter acak semacam: `Aw6EeuBmeKxyDcNl`.
  </StepperItem>
  <StepperItem title="Step 2: Masukkan Lisensi">
    Kolom untuk memasukkan plugin bisa Anda temukan di: `Plugins` > `Installed Plugins` > `Woo Cek Custom Resi` > `License`. Setelah menemukan kolom lisensi, masukkan lisensi, dan klik `Activate`
  </StepperItem>
</Stepper>
