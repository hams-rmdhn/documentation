**Tabler Login for WordPress** is a plugin that gives your website login page a new look.

By default, this plugin comes with a Honeypot to prevent bots from performing gratuitous logins or signups.

Plus, weâ€™ve added integration with Cloudflare Turnstile for optional security of your login page. The plugin includes basic features, including:

- Login page
- Register page
- Password reset page
- New password page

Some of the things you can customize include:

- Slug for login page
- Brand color
- Background color
- Brand image (logo)
- Login mascot image

The integration that we have done:

- Cloudflare Turnstile

##  General

###  Install

<Stepper>
  <StepperItem title="Step 1: Download">
    Get the plugin by purchasing in <a href="https://studioinspirasi.com/product/tabler-login-for-wordpress/?lang=en" target="_blank">Studio Inspirasi</a>, so you will get the zip file of the recent plugin update.
  </StepperItem>
  <StepperItem title="Step 2: Upload & Install">
    Upload and Install the plugin to your WordPress by navigating to your wordpress dashboard menu in: `Plugins` > `Add New` > `Upload Plugin` > `Choose File` > `Install Now`.
  </StepperItem>
  <StepperItem title="Step 3: Activate">
    After successfully install the plugin, click the `Activate` link in the process.
  </StepperItem>
</Stepper>

###  License

<Stepper>
  <StepperItem title="Step 1: Get License">
    After purchashing is successful, you'll get the license key in <a href="https://studioinspirasi.com/my-account/license-key/" target="_blank">Dashboard Page</a>. The license key is a random character like: `Aw6EeuBmeKxyDcNl`.
  </StepperItem>
  <StepperItem title="Step 2: Copy & Paste License Key">
    Copy and paste the license key in the license key field, by navigating to: `Plugins` > `Installed Plugins` > `Tabler Login for WordPress` > `License`, paste the license key in the field, and click `Activate`.
  </StepperItem>
</Stepper>
