**PPDB Announcement** adalah plugin untuk menampilkan data pengumuman berdasarkan NISN/NIM/lainnya yang dimasukkan di kolom pencarian.

Penggunaannya cukup mudah, cukup gunakan shortcode `[ppdb id=78125]` untuk menampilkan kolom pencarian di frontend. 

##  Umum

###  Instalasi

<Stepper>
  <StepperItem title="Step 1: Unduh">
    Dapatkan plugin dengan membeli di <a href="https://studioinspirasi.com/product/ppdb-announcement" target="_blank">Studio Inspirasi</a>. Setelah pembelian berhasil, Anda dapat mendapatkan file `.zip` di <a href="https://studioinspirasi.com/my-account/downloads/" target="_blank">Halaman Dasbor</a>.
  </StepperItem>
  <StepperItem title="Step 2: Unggah dan Instal">
    Unggah plugin ke website Anda melalui menu dashbor: `Plugins` > `Add New` > `Upload Plugin` > `Choose File` > `Install Now`.
  </StepperItem>
  <StepperItem title="Step 3: Aktifkan">
    Setelah berhasil melakukan instalasi, klik `Activate` untuk mengaktifkan plugin.
  </StepperItem>
</Stepper>

###  Lisensi

<Stepper>
  <StepperItem title="Step 1: Dapatkan Lisensi">
    Setelah pembelian plugin Anda berhasil, Anda akan mendapatkan lisensi di <a href="https://studioinspirasi.com/my-account/license-key/" target="_blank">Halaman Dasbor</a>. Lisensi adalah kode seperti `Aw6EeuBmeKxyDcNl`.
  </StepperItem>
  <StepperItem title="Step 2: Masukkan Lisensi">
    Kolom untuk memasukkan plugin bisa Anda temukan di: `Plugins` > `Installed Plugins` > `PPDB Announcement` > `License`. Setelah menemukan kolom lisensi, masukkan lisensi, dan klik `Activate`
  </StepperItem>
</Stepper>
