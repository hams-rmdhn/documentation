**Woo Cek Custom Resi** adalah sebuah plugin untuk menambah fitur pengecekan custom resi (airwaybill) di WooCommerce.

Plugin ini sangat cocok untuk Anda pemilik toko online pengguna WooCommerce untuk menambahkan halaman tracking untuk memudahkan pelanggan melakukan tracking pada produk yang dipesan.

<Note title="ðŸ“£ Perhatian"> 
 Plugin ini masih menggunakan `Order Data Storage` versi lawas atau disebut `Legacy` di WooCommerce. Tapi hal ini tidak mengapa karena memang fitur `HOP (High Order Performance)` di WooCommerce tergolong masih baru sehingga ketika menggunakan fitur Legacy malah lebih reliable.
</Note>

##  Umum

###  Instalasi

<Stepper>
  <StepperItem title="Step 1: Unduh">
    Dapatkan plugin dengan membeli di <a href="https://studioinspirasi.com/product/woo-cek-custom-resi/" target="_blank">Studio Inspirasi</a>. Setelah pembelian berhasil, Anda dapat mendapatkan file `.zip` di <a href="https://studioinspirasi.com/my-account/downloads/" target="_blank">Halaman Dasbor</a>.
  </StepperItem>
  <StepperItem title="Step 2: Unggah dan Instal">
    Unggah plugin ke website Anda melalui menu dashbor: `Plugins` > `Add New` > `Upload Plugin` > `Choose File` > `Install Now`.
  </StepperItem>
  <StepperItem title="Step 3: Aktifkan">
    Setelah berhasil melakukan instalasi, klik `Activate` untuk mengaktifkan plugin.
  </StepperItem>
</Stepper>

###  Lisensi

<Stepper>
  <StepperItem title="Step 1: Dapatkan Lisensi">
    Setelah pembelian plugin Anda berhasil, Anda akan mendapatkan lisensi di <a href="https://studioinspirasi.com/my-account/license-key/" target="_blank">Halaman Dasbor</a>. Lisensi adalah karakter acak semacam: `Aw6EeuBmeKxyDcNl`.
  </StepperItem>
  <StepperItem title="Step 2: Masukkan Lisensi">
    Kolom untuk memasukkan plugin bisa Anda temukan di: `Plugins` > `Installed Plugins` > `Woo Cek Custom Resi` > `License`. Setelah menemukan kolom lisensi, masukkan lisensi, dan klik `Activate`
  </StepperItem>
</Stepper>

### Halaman Pengaturan

![Halaman Pengaturan](https://si-plugins.b-cdn.net/plugins/wccr/woo-cek-custom-resi-settings.png)

Di halaman pengaturan, Anda akan mendapatkan pengaturan berikut:

- `Halaman`: mengatur basis halaman tracking cek resi
- Logo Tracking
- No. Whatsapp
- Prefix Resi
- Karakter Resi
- Jumlah Karakter Resi
- `API Key`: API Key BinderByte untuk menghubungkan resi custom dengan resi resmi ekspedisi
- Onesender API URL: API URL yang didapatkan di dasbor Onesender
- Onesender API Key: API Key yang didapatkan di dasbor Onesender untuk melakukan pengiriman pesan whatsapp otomatis ketika `Order Action` digunakan
- `Onesender Pesan Whatsapp`: Pesan whatsapp yang ingin dikirimkan melalui `Order Action`

### Detail Pengiriman

![Detail Pengiriman](https://si-plugins.b-cdn.net/plugins/wccr/woo-cek-custom-resi-detail-pengiriman.png)

Detail pengiriman adalah grup kolom yang ditampilkan di halaman order, dan berisi beberapa kolom berikut:

- `Resi Custom`: resi yang otomatis ter-generate ketika status order pelanggan berubah menjadi `On Processing`
- Kurir Ekspedisi
- Estimasi Sampai
- `Resi Kurir Ekspedisi`:  nomor resi resmi ekspedisi
